# Podología Orlan

Sitio web estático en Next.js, sin base de datos.

## Ver localmente

```bash
npm install
npm run dev
```

Abrir en el navegador:

```txt
http://localhost:3000
```

## Cambios incluidos

- Logo más grande y texto “Podología Orlan” al costado.
- Menú con letra más grande y efecto hover.
- Íconos de servicios más relacionados con cada categoría.
- Sección de opiniones estilo Google Reviews.
- Fondo celeste suave, más relacionado con salud.
- Tipografía más grande, clara y legible para adultos mayores.
- Diseño responsive para celular.

## Importante

Las opiniones son textos de maqueta con estilo visual tipo Google. Cuando tengas reseñas reales, reemplaza los nombres y comentarios en `app/page.js`.
